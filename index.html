accessibility.plugin.js

the simplest and most practical plugin for creating accessibility buttons

option 1 - cloning the repo

run the following command in your terminal (the plugin files are located in the /plugin folder)

git clone https://github.com/gustavocoimbradev/accessibility-plugin-js.git

option 2 - using the CDN

CSS

<link href="https://gustavocoimbradev.github.io/accessibility-plugin-js/plugin/css/accessibility.plugin.css" rel="stylesheet">

JS

<script src="https://gustavocoimbradev.github.io/accessibility-plugin-js/plugin/js/accessibility.plugin.js"></script>

basic usage

<div class="js-accessibility accessibility">

    <button class="js-increase increase">A+</button>

    <button class="js-decrease decrease">a-</button>

    <button class="js-contrast contrast">contrast</button>

</div>

** feel free to change the text of the buttons and style the buttons as you wish using the classes .accessibility, .increase, .decrease and .contrast
